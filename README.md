# 🎓 KRS & KHS Management System

An academic information system designed to manage **Study Plan Cards (KRS)** and **Study Result Cards (KHS)** in an integrated and efficient way.

This desktop-based application helps streamline academic administration, reduce errors, and improve accessibility for students, lecturers, and administrators.

---

## 🚀 Features

- 🔐 Multi-role authentication (Admin, Lecturer, Student)  
- 📚 Student and lecturer data management  
- 🗓️ Course and schedule management  
- 📝 KRS (course registration) by students  
- ✅ KRS approval by lecturers (Approve / Reject)  
- 📊 KHS (grade report) visualization  
- 📄 Export KRS & KHS to PDF  

---

## 🧑‍💻 Technologies Used

- 💻 Visual Basic .NET (VB.NET)  
- 🛠️ Microsoft Visual Studio  
- 🗄️ SQL Server  
- 🧪 Black Box Testing  

---

## ⚙️ Installation & Setup

1. Clone this repository
2. Open the `.sln` file using Microsoft Visual Studio
3. Make sure:
   - SQL Server is installed
   - Database is properly configured
4. Run the project (Ctrl + F5)

---

## 🧪 Testing
System testing was conducted using **Black Box Testing**, ensuring all functionalities run as expected.
